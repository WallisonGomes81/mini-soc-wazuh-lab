# 01 – Ambiente do Laboratório

## Objetivo
Este laboratório tem como objetivo estudar e praticar o uso do Wazuh
como SIEM, simulando atividades básicas de um SOC (Security Operations Center).

## Ambiente
- Wazuh Server
- 1 VM Linux (Ubuntu ou Kali) como agente
- 1 VM Windows como agente
- Máquina atacante para simulação de eventos

## Ferramentas
- Wazuh
- VirtualBox / VMware
- Kali Linux
- Windows
