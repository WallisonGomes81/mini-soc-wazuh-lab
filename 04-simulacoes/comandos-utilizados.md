# Comandos Utilizados nas Simulações

## Linux
- Tentativas de acesso SSH
- Execução de ferramentas de teste de autenticação
- Comandos básicos de administração para geração de logs

## Windows
- Criação de usuário local
- Tentativas de login inválidas
- Logoff e troca de usuários

## Observação
Todos os comandos e ações foram executados em ambiente controlado,
exclusivamente para fins educacionais.
